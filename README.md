# unemployment-analysis
# Unemployment Analysis in India 🇮🇳

## 📌 Description
This project analyzes unemployment data in India using Python. It explores trends, correlations, and regional differences using data visualization techniques.

## 📊 Features
- Data cleaning and preprocessing
- Handling missing values
- Correlation heatmap using Seaborn
- Distribution analysis using histograms
- Interactive visualization using Plotly (Sunburst chart)

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## 📂 Dataset
The dataset is loaded directly from GitHub:
https://raw.githubusercontent.com/hemanthkumarsn2002/unemployment-analysis/main/unemployment.csv
